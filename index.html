🧳 行李准备详细清单

衣物类

  • 冲锋衣 / 薄羽绒服 — 每人1件，早晚只有5-10℃，垭口风大，必备！
  • 长袖T恤 / 速干衣 — 每人3-4件，白天防晒透气
  • 薄外套 / 防晒衣 — 每人1件，车内或阴天穿
  • 长裤（运动裤 / 牛仔裤） — 每人2条，草原蚊虫多且晒，不建议短裤
  • 保暖裤 / 秋裤 — 每人1条，阿坝、莲宝叶则早晚用得上
  • 舒适徒步鞋 — 每人1双，九寨沟+扎尕那日均1.5万步，鞋底要软
  • 内衣裤 + 袜子 — 每人5-6套，高原干燥不易干
  • 遮阳帽 + 墨镜 — 每人1副，高原紫外线极强，不戴会晒伤
  • 围巾 / 薄披肩 — 每人1条，防风+拍照道具+车内盖腿
  • 拖鞋 — 每人1双，民宿/酒店用

药品类

  • 红景天 — 提前7天开始服用，全家一起
  • 便携氧气瓶 — 每人2-3瓶，成都/九寨沟购买，莲宝叶则+洛克之路用
  • 布洛芬 / 对乙酰氨基酚 — 高反头痛最有效
  • 葡萄糖口服液 — 快速补充能量+缓解高反
  • 晕车药（茶苯海明） — 盘山路多，容易晕车
  • 肠胃药（蒙脱石散 / 黄连素） — 水土不服
  • 感冒药 + 退烧药 — 高原感冒恢复慢
  • 日常慢病药 — 比行程多带3天量，分两处放
  • 创可贴 + 碘伏棉签 — 小伤口处理
  • 体温计 — 电子版，随时监测

证件与支付

  • 身份证 — 每人随身携带，住宿+景区实名
  • 驾驶证 + 行驶证 — 司机携带，电子驾照备一份
  • 银行卡 + 现金 — 500-1000元，沿途部分加油站/牧区只收现金
  • 手机 + 充电宝 — 每人，高原耗电快，2万毫安充电宝必备
  • 紧急联系人卡片 — 写上基础病史和过敏药物
  • 门票预约截图 — 九寨沟+三星堆预约码提前截图保存

洗护与防晒

  • 防晒霜（SPF50+ PA+++） — 每人，高原必涂，每2小时补一次
  • 润唇膏（带防晒） — 每人，嘴唇极易干裂
  • 保湿乳液 / 面霜 — 每人，高原极度干燥
  • 洗发水 + 沐浴露（旅行装） — 部分民宿不提供
  • 牙刷 + 牙膏 + 毛巾 — 每人，民宿条件有限
  • 湿巾 + 纸巾（大包） — 沿途厕所条件差，大量备
  • 免洗洗手液 — 车上/景区吃饭前用

车载与路上

  • 车载手机支架 — 导航必备
  • 车载充电器（多口） — 同时充多部手机
  • 保温杯（大容量） — 每人，高原喝热水非常重要
  • 矿泉水（1箱） — 成都购买，沿途补水
  • 高热量零食 — 巧克力/牛肉干/饼干，洛克之路无补给
  • 水果（苹果 / 黄瓜） — 补充维生素
  • 垃圾袋 — 几个，车上垃圾不乱扔
  • 颈枕 + 腰靠 — 每人，长途车程多，腰靠对腰椎很友好
  • 折叠雨伞 / 雨衣 — 7月雨季，草原阵雨多
  • 离线地图 — 提前下载甘肃+四川离线地图

其他实用小件

  • 折叠小板凳 / 坐垫 — 景区排队或草原休息用
  • 便携烧水壶 — 高原酒店水壶不干净，自己带
  • 晾衣绳 + 折叠衣架 — 民宿晾衣服用
  • 指甲刀 — 小物件但实用
  • 日记本 + 笔 — 记录旅行点滴
  • 扑克牌 — 晚上在酒店玩
